

По причине крайне низкой скорости выполнения rest-запросов, особенно при первоначальной выборке
данных (2000 аттрибутов требуют 2000 запросов), следует все неизменяемые данные загрузить в бд и пользоваться
впоследствие ими из бд. Загрузков данных разбить на множестов потоков, например 2000 запросов разбить на 20 потоков
по 100 запросов - это значительно ускорит скорость загрузки данных.
